# Roxy-irl

An AI assistant based on Roxanne Wolf (FNAF Security Breach), built to run either locally or via an API (configurable in `config.json`). This project is a foundational test designed for further optimization, refinement, and custom coding.

## Prerequisites

Before setting up the project, ensure you have the following:

* **Operating System:** Windows 10 or 11
* **Python:** Python 3.9 or higher installed
* **Terminal:** PowerShell or Command Prompt (CMD)

## Installation

1. **Download the project:** Get the latest stable version from the [Releases Page](https://github.com/karakulakdede32-ui/Roxy-irl/releases) and extract the files.
2. **Install dependencies:** Open your terminal in the project directory and run:
   ```bash
   pip install -r requirements.txt
